# Бонусная Программа
## Установка
1.  Клонирование репозитория:
    ```bash
    git clone <репозиторий>
    cd lr9
    ```
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
## Запуск
```bash
python app.py
```

## Важно
Необходиммо создатьв в проекте файл ".env", в котором будет храниться ключ:
```bash
SECRET_KEY=your_secret_key
```
